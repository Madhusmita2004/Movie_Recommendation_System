# Movie Recommendation System 🎬

This is a Hybrid Movie Recommendation System built using:
- Content-Based Filtering
- Collaborative Filtering (User-Based)
- Matrix Factorization using ALS (Alternating Least Squares)

It uses the MovieLens dataset to suggest personalized movie recommendations.

## 🔧 Features
- Hybrid recommendation combining different algorithms
- Flask backend API
- HTML frontend interface
- Model serialized with pickle for easy deployment

## 🚀 How to Run the Project

### 1. Clone the Repository
```bash
git clone https://github.com/Madhusmita2004/Movie_Recommendation_System.git
cd Movie_Recommendation_System

2. Install Dependencies

pip install -r requirements.txt

3. Run Flask App

python app.py

4. View in Browser
Go to: http://127.0.0.1:5000/

5. Frontend (Optional)
Open index.html and use it to submit your movie name, user ID, and get recommendations.

📁 Files
- app.py - Flask backend API

- index.html - Frontend UI

- *.pkl - Pickle files for models and data

- README.md - Project details
